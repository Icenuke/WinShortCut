# --==[ ExtractStrings ]==--
## Description:
> ExtractStrings use the software nammed [Strings](https://live.sysinternals.com/) <br>
> To extract all printable strings from a file.<br>
> The shortcut can be use on many files to automatized the extract
## How to install:
> 1. Download Strings.exe from [sysinternals](https://live.sysinternals.com/) 
> 2. Add strings in C:\Windows\ or C:\Windows\System32\
> 3. Launch the '.reg' file
## How to use:
> *Possible with many files*
> 1. Right click on a file
> 2. Click in ExtractString
> 3. Read the export
